> A curated list of awesome tools, resources, services, and guides for building websites and apps with the JAMstack architecture.

JAMstack (JavaScript, APIs, and Markup) is a modern web development architecture based on client-side JavaScript, reusable APIs, and prebuilt Markup. This list brings together everything you need to build fast, scalable, and secure websites.

## Static Site Generators

- [Next.js](https://nextjs.org/) – React-based hybrid static and server-rendered site framework.
- [Gatsby](https://www.gatsbyjs.com/) – React-based SSG with plugin ecosystem.
- [Hugo](https://gohugo.io/) – Fast, Go-powered static site generator.
- [Eleventy](https://www.11ty.dev/) – Simpler static site generator with great flexibility.
- [Jekyll](https://jekyllrb.com/) – Blog-aware SSG popularized by GitHub Pages.
- [Nuxt](https://nuxt.com/) – Vue.js framework that supports static generation.
